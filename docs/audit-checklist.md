# AI 应用安全审计检查清单 v0.1

## 身份认证

- [ ] 登录接口是否防止弱口令
- [ ] Token 是否有签名
- [ ] Token 是否可被篡改
- [ ] 退出登录或 Token 失效机制是否设计

## 授权与越权

- [ ] 普通用户是否只能访问自己的文档
- [ ] 不同租户之间是否隔离
- [ ] 管理员接口是否强制管理员角色
- [ ] 修改 `document_id` 是否能访问他人数据

## 文档资产

- [ ] 文档是否记录上传者
- [ ] 文档是否记录租户
- [ ] 文档内容是否可能包含敏感信息
- [ ] 文档是否会被后续 RAG 无权限召回

## 日志与敏感信息

- [ ] 是否记录明文密码
- [ ] 是否记录完整 Token
- [ ] 是否记录完整文档内容
- [ ] 是否需要脱敏邮箱、手机号、密钥

## 后续 RAG 安全

- [ ] 检索前是否做租户过滤
- [ ] 检索结果是否可能跨用户泄露
- [ ] 恶意文档是否能提示注入
- [ ] 模型输出是否会被当作权限判断
- [ ] 模型输出是否进入 SQL、HTML、命令或工具调用
